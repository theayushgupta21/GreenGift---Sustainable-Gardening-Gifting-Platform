# 🌱 GreenGift — Sustainable Gardening & Eco-Gifting Platform

> “Gift Green, Grow India” 🇮🇳

GreenGift is a modern, mobile-first sustainable gardening and eco-gifting platform built using **Next.js 14**, **Tailwind CSS**, **shadcn/ui**, and **Zustand**.

The platform helps users:
- 🌿 Buy plants, pots, seeds, and gardening tools
- 🎁 Send eco-friendly gifts
- ♻️ Support sustainability through seed balls
- 🌳 Participate in Green India plantation initiatives
- 📱 Order easily through WhatsApp

Designed especially for:
- Rural & low-tech users
- Mobile-first customers
- Elderly users
- Eco-conscious buyers
